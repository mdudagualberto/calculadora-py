# 🧮 Calculadora

> Calculadora simples para dois números, rodando direto no terminal!

---

## 📖 Sobre o Projeto

O usuário digita dois números e escolhe qual operação deseja realizar. O programa então calcula e exibe o resultado na tela.

Operações disponíveis:
- ➕ Adição
- ➖ Subtração
- ✖️ Multiplicação
- ➗ Divisão

---

## 🎮 Como Funciona (Exemplo)

```
Digite o primeiro número: 10
Digite o segundo número: 5
Escolha a operação (+, -, *, /): *

Resultado: 10 * 5 = 50
```

---

## 🚀 Como Rodar

### Pré-requisitos

Você precisa ter o **Python** instalado na sua máquina.

> Não tem Python? Baixe em: https://www.python.org/downloads/

Para verificar se já está instalado, abra o terminal e digite:

```bash
python --version
```

Se aparecer algo como `Python 3.x.x`, está tudo certo!

---

### Passo a Passo

**1. Clone o repositório:**

```bash
git clone https://github.com/mdudagualberto/calculadora-py.git
```

**2. Entre na pasta do projeto:**

```bash
cd calculadora-py`

**3. Execute o programa:**

```bash
python calculadora.py
```
